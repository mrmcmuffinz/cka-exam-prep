# TLS and Certificates Assignment 2: Certificates API and kubeconfig

This is the second of three assignments covering TLS and certificates in Kubernetes. This assignment focuses on the Kubernetes Certificates API for automated CSR workflows and kubeconfig file structure and management.

## Assignment Overview

While manual certificate creation works, Kubernetes provides the Certificates API for automated CSR approval workflows. This assignment teaches how to use CertificateSigningRequest resources and how to configure kubeconfig files for certificate-based authentication.
