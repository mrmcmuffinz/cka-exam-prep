# Ingress and Gateway API Assignment 2: Advanced Ingress and TLS

This is the second of three assignments covering Ingress and Gateway API. This assignment focuses on Ingress annotations, rewrite-target, TLS termination, certificate management, and advanced routing patterns. Basic Ingress is assumed from assignment 1. Gateway API is covered in assignment 3.

## Prerequisites

- exercises/ingress-and-gateway-api/assignment-1 (Ingress Fundamentals)

## What You Will Learn

Advanced Ingress configuration including controller-specific annotations, URL path rewriting, TLS termination with certificates, creating TLS secrets, and complex routing scenarios.

## Estimated Time

4 to 6 hours for the tutorial and all 15 exercises.

## Cluster Requirements

Same as assignment 1: kind cluster with nginx-ingress installed.

## Difficulty Progression

**Level 1:** Annotations including ssl-redirect and common configurations.

**Level 2:** Rewrite-target and TLS with self-signed certificates.

**Level 3:** Debugging advanced Ingress issues.

**Level 4:** Complex configurations with multiple TLS hosts.

**Level 5:** Production patterns and migration scenarios.
