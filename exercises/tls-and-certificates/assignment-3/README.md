# TLS and Certificates Assignment 3: Certificate Troubleshooting

This is the third and final assignment covering TLS and certificates in Kubernetes. This assignment focuses on diagnosing and resolving certificate-related issues including expiration, subject/issuer mismatches, wrong CA, permission issues, and certificate renewal.

## Assignment Overview

Certificate problems cause cryptic errors and can bring down clusters. This assignment teaches systematic troubleshooting: checking expiration, validating certificate chains, diagnosing common errors, and performing certificate renewal.
